# 产品需求文档（PRD）

## 一、项目背景

Tom Blog 是一个个人技术博客，用于发布技术文章、记录学习笔记。分为对外展示页和内部管理后台两个端。

---

## 二、角色

| 角色 | 说明 |
|------|------|
| 访客 | 无需登录，可浏览文章 |
| 管理员 | 登录后台，管理文章、分类、标签 |

---

## 三、展示页功能

### 3.1 首页
- 文章列表，按发布时间倒序排列
- 支持按分类、标签筛选
- 分页展示

### 3.2 文章详情页
- 渲染 Markdown 内容
- 代码块语法高亮
- 显示文章标题、发布时间、分类、标签

### 3.3 主题切换
- 支持明/暗两种主题
- 用户切换后记住偏好（localStorage）

### 3.4 关于页
- 展示博主简介，内容由管理员在配置中维护

---

## 四、后台管理功能

### 4.1 登录
- 账号密码登录，返回 JWT token
- token 过期后自动跳转登录页

### 4.2 文章管理
- 文章列表（支持按标题搜索、按分类筛选）
- 新建文章（标题、封面图、分类、标签、Markdown 正文、是否发布）
- 编辑文章
- 删除文章（软删除）
- 发布 / 下线切换

### 4.3 分类管理
- 分类列表
- 新增、编辑、删除分类

### 4.4 标签管理
- 标签列表
- 新增、编辑、删除标签

### 4.5 文件上传
- 上传图片到腾讯云 COS
- 返回可访问的图片 URL

---
